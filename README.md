# Eternal Trivium — Series Codex

The living worldbuilding reference for **Eternal Trivium**, a fantasy franchise set in the
spirit-energy world of **Vi'talitas**. The primary medium is **books**, with a tabletop RPG
and an animated series planned as downstream adaptations that follow the novels.

## The world at a glance

- **Animancy** — the magic system. Power drawn from **memory**, paid in lost feeling
  (greying) rather than numeric cost. Sustained by **Savvy** in the **Recall State**;
  fails into **Mania** when a memory is Recalled too deeply.
- **The Nine Brands** — the disciplines, each guarded by one of the **Nine Divine Aspects**,
  each resonating with a family of emotion and surfacing **Signets** at a fixed body location.
- **V.I.T.A.L.I.T.A.S.** — the nine core attributes (the game-layer stat system), which
  spell the world's name.
- **The three Paramounts** — A'thule (Dark Matter), A'thesis (Matter), A'kana (Anti-Matter),
  who sleeps as the Great Mother Tree at the Eye of the Aegis.
- **Protagonists** — the Elloyd siblings: Omin, Amoz, and I'onnie.
- **Antagonist** — Silas, the only Neversoul fully Stigmatized yet self-possessed.
- **Factions** — the Trauma Contingency Troupe (T.C.T.), the A.A.A. Institute, the
  Aggregate Regency, and the Undertrade Exchange.

## Working convention

- Confirmed canon is marked **[LOCKED]**; open questions **[OPEN]**.
- Edit the **Markdown** first; the HTML mirrors it.
- Canon is never guessed at — decisions are made deliberately and their implications
  flagged before locking.

## Updating the live site

1. Replace `index.html` (and the `.md`) with the latest version.
2. Commit and push.
3. Wait ~1 minute for GitHub Pages to rebuild, then hard-refresh (Ctrl/Cmd+Shift+R).
4. When sharing a fresh update, append a version tag to the link (e.g. `?v=2`) to bypass caching.

## Status

Core systems — cosmology, the magic system, the full casting cycle (Signature, the Recall
State, emotion-driven Memory, and Mania), the nine attributes, time & currency, and the
character roster — are substantially locked. A prototype character sheet exists. The next
phase is testing the world in actual prose.
